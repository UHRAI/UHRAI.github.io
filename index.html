Meet the Team

Inspiration

"The Maze" project was born out of a personal fascination with the intersection of maze-solving algorithms and immersive 3D graphics. The desire to create a captivating gaming experience, where players navigate through a visually stimulating maze, served as the driving force behind this endeavor. The inspiration for this project stems from a childhood love for maze puzzles and a deep-seated curiosity about the technical aspects of transforming a 2D map into a navigable 3D world using ray casting.

Timeline

The journey commenced as a personal exploration, evolving into a passion project that combined programming prowess with creative design. While the initial spark ignited on Dec 1, 2023, at 4:00 AM, the development continued to evolve, culminating in a refined project that exceeds initial expectations.

Holberton School Portfolio Project: "The Maze" stands as a proud Portfolio Project for Holberton School. It not only showcases technical proficiency but also demonstrates a commitment to creating engaging and interactive experiences. This project holds significance as a testament to the skills and creativity cultivated during the Holberton School journey.
